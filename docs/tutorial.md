# ベイズ的サイコロ予測チュートリアル

## 1. この教材で何を学ぶか
この教材では、歪んだサイコロをブラックボックスとして扱い、観測された出目だけから各面の出やすさを推定します。ポイントは、**答えを当てること**よりも、**どの程度の自信を持って推定しているか**を理解することです。

### 確認問題
1. この教材で重視しているのは、単に正解の面を当てることだけでしょうか。
2. なぜサイコロの内部確率を見ずに、観測結果だけで学ぶ設定にしているのでしょうか。

### 答え
1. いいえ。重要なのは、どの面が出やすいかの予測に加えて、その予測にどれほど自信が持てるかを理解することです。
2. 現実の多くの問題では、内部の仕組みは直接見えず、観測データだけが手に入ります。その状況を学ぶために、ブラックボックス設定を採用しています。

## 2. 最初に試すこと
### 基本デモ
次を実行すると、観測データと事後予測確率が表示されます。

python run_demo.py

見るべき点は次の通りです。
- Observed rolls: 実際に見えた出目列
- Posterior counts: 各面を何回観測したか
- Posterior predictive probabilities: 次に各面が出ると予測する確率
- Most likely next face: 現時点で最も出やすいと推定される面

### 収束確認
次を実行すると、観測が増えるにつれて予測がどう変化するかを確認できます。

python analyze_convergence.py

ここでは、少数サンプルでは推定が揺れやすく、観測が増えるほど安定していく様子に注目してください。

### 確認問題
1. Basic demo の出力で、各面が何回観測されたかを見る項目はどれですか。
2. Posterior predictive probabilities は、何を表していますか。
3. 収束確認では、観測回数が増えるとどのような変化を期待しますか。

### 答え
1. Posterior counts です。ここを見ると、どの面が何回出たかを確認できます。
2. 今までの観測を踏まえて、次の1回で各面が出ると予測する確率です。
3. 少数サンプルで見られた大きな揺れが徐々に小さくなり、予測確率が安定していくことを期待します。

## 3. 不確実性とは何か
ベイズ統計では、単に「6の目が最も出やすい」と言うだけでなく、**その判断にどれくらい確信があるか**も扱います。

- 観測が少ないと不確実性は大きい
- 観測が増えると不確実性は小さくなる
- 事前分布は、観測が少ないときほど影響しやすい

### 確認問題
1. 観測が 10 回しかない場合と 100 回ある場合では、一般にどちらのほうが不確実性は大きいですか。
2. ある面が現在もっとも出やすいと予測されていても、不確実性が大きいとはどういう状態ですか。
3. 事前分布の影響が強く出やすいのは、観測が多いときと少ないときのどちらですか。

### 答え
1. 一般には 10 回しかない場合のほうが不確実性は大きいです。データが少ないため、推定のぶれが大きくなります。
2. 現時点ではその面が有力に見えていても、まだ十分なデータがなく、別の面が有力になる可能性が残っている状態です。
3. 観測が少ないときです。データよりも事前の仮定が相対的に効きやすいためです。

## 4. 自分で試す実験
1. サイコロの確率を変える
2. 観測回数を 30 から 100 に増やす
3. どの面が一番推定しやすいかを比べる
4. 不確実性の変化を読む
