# ベイズ的サイコロ予測チュートリアル

## 1. この教材で何を学ぶか
この教材では、歪んだサイコロをブラックボックスとして扱い、観測された出目だけから各面の出やすさを推定します。ポイントは、**答えを当てること**よりも、**どの程度の自信を持って推定しているか**を理解することです。

## 2. 最初に試すこと
### 基本デモ
次を実行すると、観測データと事後予測確率が表示されます。

python run_demo.py

見るべき点は次の通りです。
- Observed rolls: 実際に見えた出目列
- Posterior counts: 各面を何回観測したか
- Posterior predictive probabilities: 次に各面が出ると予測する確率
- Most likely next face: 現時点で最も出やすいと推定される面

### 収束確認
次を実行すると、観測が増えるにつれて予測がどう変化するかを確認できます。

python analyze_convergence.py

ここでは、少数サンプルでは推定が揺れやすく、観測が増えるほど安定していく様子に注目してください。

## 3. 不確実性とは何か
ベイズ統計では、単に「6の目が最も出やすい」と言うだけでなく、**その判断にどれくらい確信があるか**も扱います。

- 観測が少ないと不確実性は大きい
- 観測が増えると不確実性は小さくなる
- 事前分布は、観測が少ないときほど影響しやすい

## 4. 自分で試す実験
1. サイコロの確率を変える
2. 観測回数を 30 から 100 に増やす
3. どの面が一番推定しやすいかを比べる
4. 不確実性の変化を読む

- 事後予測シミュレーション
- 公平なサイコロとの比較
- 時間によって歪みが変わるケースの検知
